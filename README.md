# 🎬 Bot Convertidor de Videos para Telegram

Bot profesional para convertir y comprimir videos en Telegram con sistema de colas, base de datos y barra de progreso en tiempo real.

## ✨ Características

- ✅ Conversión de videos a MP4 con compresión inteligente
- ✅ Sistema de colas avanzado con múltiples procesos
- ✅ Barra de progreso en tiempo real
- ✅ Base de datos JSON para estadísticas e historial
- ✅ Funcionamiento en grupos y chats privados
- ✅ Comandos de administración completos
- ✅ Exportación e importación de base de datos con `/exportdb` y `/importdb`
- ✅ Cancelación de conversiones con botón y comando `/cancelar`
- ✅ Soporte para thumbnails automáticos
- ✅ Interfaz en español
